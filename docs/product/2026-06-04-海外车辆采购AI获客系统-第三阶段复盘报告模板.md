## 9. Go / 重跑 / 暂停判定

### 9.1 Go：进入下一阶段扩大运行

建议 Go 必须同时满足：

- [ ] 有效客户承接率达到 60% 或已证明稳定提升。
- [ ] 补全成功率达到 50% 或补全结果被客服 / 销售认为可用。
- [ ] 字段采纳率达到 40% 或主要关键字段采纳质量稳定。
- [ ] 客户晋级污染事件数为 0。
- [ ] 风险违规事件数为 0，风险门禁未失效。
- [ ] 自动社交私信、自动加好友、登录后批量采集、反爬规避、非公开数据抓取均为 0。
- [ ] 清洗采纳率达到 40% 或能明显减少 Watch / Invalid / 重复线索堆积。
- [ ] 客服和销售愿意继续承接。
- [ ] C 级报价、合同、付款、物流、清关、交付周期前的合规复核路径明确。
- [ ] LLM 成本在预算内，且失败重试没有造成成本不可控。

### 9.2 重跑：调整后再跑一轮

出现以下情况之一，且无严重合规风险时，建议重跑：

- [ ] 有效客户承接率未达标，但客服 / 销售认为线索可通过补全或交付格式优化改善。
- [ ] 补全成功率不足，但失败集中在可修复的 prompt、关键词、渠道范围或 schema 问题。
- [ ] 字段采纳率不足，但主要因为证据摘要、字段说明或人工审核体验不清晰。
- [ ] 清洗采纳率不足，但建议类型或置信度规则有优化空间。
- [ ] LLM 成本偏高，但可通过配额、读取链接数、搜索 query 数、模型选择或重试策略控制。
- [ ] 销售承接反馈一般，但不是明确拒绝继续承接。
- [ ] 风险事件可控，未出现风险门禁失效或违规自动化动作。

### 9.3 暂停：暂停当前方向或市场

出现以下任一情况，应暂停：

- [ ] 客户晋级污染：未满足准入条件、无来源证据、勿扰、Forbidden、AI 编造字段进入 core。
- [ ] 销售不愿承接：出口销售明确认为客户质量、车型意向、交易路径或跟进价值不足。
- [ ] 风险门禁失效：自动触达、自动加好友、登录后批量采集、反爬规避、非公开数据抓取、Forbidden 关键证据、High 未审核触达等任一硬边界失效。
- [ ] LLM 成本不可控：成本持续超预算，且无法通过配额、模型、重试、读取链接数或 prompt 优化控制。
- [ ] 有效客户承接率长期低于 30%，且客服 / 销售反馈无明显改进空间。
- [ ] 补全成功率长期低于 20%，且字段采纳质量不足以支撑晋级。
- [ ] 清洗建议大量误判，导致重复归并、Watch 恢复或 Invalid 确认不可被人工信任。
- [ ] 出现客户投诉、平台封禁、明显违规采集或外部合规风险。
- [ ] C 级客户交易路径、付款、物流、清关或合规复核不可行。
