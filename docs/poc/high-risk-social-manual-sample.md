# 高风险社媒人工小样本研究表

创建日期：2026-05-27  
适用范围：VK / Odnoklassniki / TikTok / Max 等高风险社媒或通讯平台的政策研究与人工小样本评估。

## 1. 结论

本研究表不作为正式线索采集结果，不进入触达队列，不交付客服或销售。

允许用途：

- 政策研究。
- 人工小样本查看公开页面。
- 记录平台是否公开展示官网、邮箱、电话、WhatsApp、Telegram 等可迁移到 Low/Medium 来源复核的信息。

禁止用途：

- 自动采集 VK/OK/TikTok/Max 页面。
- 登录后批量采集。
- 自动关注、自动私信、自动加好友。
- 抓取评论、粉丝、好友、互动关系或用户画像。
- 绕过验证码、访问限制或平台风控。
- 将 High 风险平台页直接进入 CRM 触达队列。

## 4. 人工小样本 SOP

1. 每个平台每次最多人工查看 3-5 个公开主页。
2. 不登录，不使用异常账号，不绕过访问限制。
3. 不记录粉丝、评论、好友、点赞、私信或任何关系链。
4. 如果公开主页展示官网，优先跳转官网并从官网采集。
5. 如果公开主页展示邮箱、电话、WhatsApp 或 Telegram，只能记录为“待官网复核”。
6. 任何样本不得进入客服/销售触达队列。
7. 若样本有明确拒绝商务联系说明，标记勿扰。
8. 每次研究后由合规负责人复核是否继续保留该渠道。

## 5. Go / No-Go 建议

- VK/OK/TikTok/Max 当前结论：No-Go for automated collection。
- 可作为辅助发现渠道：仅人工小样本发现官网或公开商务联系方式。
- 可进入正式 PoC 的唯一方式：从 High 平台跳转到客户官网、公开目录或地图公开页后，在 Low/Medium 来源重新取证。
