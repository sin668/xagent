## 9. Go / 重跑 / 暂停判定

### 9.1 Go：继续进入 MVP

必须同时满足：

- [ ] 至少 3 个可持续低/中风险渠道。
- [ ] 100-200 条线索中，B 级比例达到 30%-50% 或接近目标且趋势明确。
- [ ] 小样本触达有真实回复或明确负反馈。
- [ ] 无账号封禁、平台投诉、明显违规采集或批量骚扰。
- [ ] 客服/销售认为字段、话术和交付方式可用。
- [ ] C 级报价/合同前合规复核路径明确。

### 9.2 重跑 PoC

满足以下情况之一：

- [ ] 渠道或关键词需要调整，但仍看到有效线索。
- [ ] B 级比例未达标但高于 20%，且有优化空间。
- [ ] 触达反馈不足，但客服/销售认为线索可用。
- [ ] 风险可控，但话术、地区、车型或客户画像需要调整。

### 9.3 暂停俄罗斯方向

出现以下任一情况，应暂停或转向：

- [ ] 100-200 条线索中，B 级线索低于 20%。
- [ ] 小样本触达几乎无人回复，或多数明确拒绝。
- [ ] 出现投诉、封禁、账号限制、访问异常或明显违规风险。
- [ ] 销售团队不愿承接。
- [ ] 客户有兴趣但贸易、支付、物流或清关路径不可行。
- [ ] High/Forbidden 渠道成为主要线索来源，且无法合规替代。
