# Story E4-S3：半自动邮件/官网表单发送

## 基本信息

- Epic：E4 AI 触达助手
- Sprint：Backlog
- 优先级：P1
- 状态：Draft
- 负责人建议：产品负责人 + 合规/风控 + 后端工程

## 用户故事

作为客服，我希望在人工确认后通过低风险渠道发送邮件或官网表单。

## 业务价值

在合规可控前提下提升操作效率。

## 进入条件

- PoC 回复率和合规边界明确。
- Low 风险渠道稳定。
- 勿扰机制和触达审计已经上线。

## 任务清单

- [ ] 明确只允许 Low 风险渠道。
- [ ] 发送前展示完整草稿和风险检查。
- [ ] 人工确认后发送。
- [ ] 发送后自动创建触达记录。
- [ ] 失败时保留错误原因。

## 验收标准

- 只允许 Low 风险渠道。
- 发送前必须展示完整草稿和风险检查。
- 发送后自动创建触达记录。
- 失败时保留错误原因。

## 非目标

- 不接入 WhatsApp/Telegram/VK 自动发送。
- 不做自动社交私信。

## QA / 风控检查

- [ ] 任何自动发送能力上线前必须经过合规复核。
- [ ] 拒绝联系客户必须被排除。
