# Story E6-S3：C 级线索合规复核

## 基本信息

- Epic：E6 合规与风险护栏
- Sprint：Sprint 5 MVP 业务闭环
- 优先级：P0
- 状态：Done
- 负责人建议：合规/风控 + 出口销售 + 后端工程

## 用户故事

作为出口销售，我希望 C 级线索在报价或合同前触发合规复核。

## 业务价值

控制俄罗斯贸易、支付、物流、清关风险。

## 依赖

- E3-S3 移动端线索详情
- E5-S2 线索与车源匹配

## 任务清单

- [x] 为 C 级线索增加合规复核状态。
- [x] 支持记录复核人、时间、结论和风险备注。
- [x] 未复核时阻断“已报价”状态。
- [x] 在详情页展示复核状态。
- [x] 在管理后台展示待复核队列。
- [x] AI 仅提示风险，不生成法律结论。

## 验收标准

- C 级线索进入报价/合同动作前显示合规复核状态。
- 未复核时不能标记为已报价。
- 复核记录包含复核人、时间、结论和风险备注。
- AI 只能提示风险，不能替代复核结论。

## 非目标

- 不做法律意见自动生成。

## QA / 风控检查

- [x] 未复核 C 级线索不能进入报价完成状态。
- [x] 复核记录不可被普通用户覆盖。

## 实现记录

- 后端新增 `/compliance/reviews/pending`、`/compliance/customers/{customer_id}/status`、`/compliance/customers/{customer_id}/review`、`/compliance/customers/{customer_id}/mark-quoted`。
- C 级线索会自动生成待复核记录；未通过合规复核前，报价状态更新被阻断。
- 复核记录包含复核人、复核时间、结论、原因、风险备注；普通销售角色提交复核会返回 403。
- 移动端线索详情新增合规复核区域，展示复核状态、复核人、风险备注、AI 风险提示和报价阻断状态。
- AI 文案固定为风险提示，不输出法律结论。

## 验收结果

- 通过：C 级线索进入报价动作前显示合规复核状态。
- 通过：未复核 C 级线索不能标记为已报价。
- 通过：合规复核通过后可标记为已报价。
- 通过：复核记录包含复核人、时间、结论和风险备注。
- 通过：普通用户不可覆盖复核记录。
- 通过：移动端详情页展示复核状态，非 C 级线索不显示错误阻断。
