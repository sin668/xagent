## 实现记录

- 新增 `GET /dashboard/admin-overview`，聚合 E7 渠道线索指标、E7 SLA 指标、团队队列和 AI 风险阻断日志。
- 后台首页切换为总览看板，展示候选线索、B/C 线索、回复率、SLA 风险、渠道产出、运营/客服/销售队列、风险事件与阻断原因。
- High/Forbidden 渠道只展示研究/阻断状态，不进入可投入建议；勿扰客户不进入今日队列。
- 阻断任务来源于 `ai_audit_logs.risk_blocked = true`，展示来源链接和 `risk_block_reason`。
- 跳转能力在当前静态后台中以侧边栏信息架构承接；真实路由拆页将在 E8 后续 Story 中完善。
