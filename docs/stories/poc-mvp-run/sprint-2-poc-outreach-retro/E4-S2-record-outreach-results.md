- Epic：E4 AI 触达助手

## 用户故事

作为客服，我希望记录每次触达渠道、话术版本、发送人、发送时间和回复状态。

## 业务价值

支持回复率复盘、SLA 管理和客户交接。

## 依赖

- E4-S1 生成俄语触达草稿

- [x] 记录发送人、发送时间、触达渠道、话术版本。
- [x] 客户拒绝后标记勿扰。
- [x] 将回复内容摘要回填到客户线索表。

## 验收标准

- 支持状态：未发送、已发送、已回复、拒绝、无回复、错误联系方式。
- 客户拒绝后可一键标记勿扰。
- 勿扰客户不再出现在触达任务中。
- 触达记录可在客户详情页查看。
