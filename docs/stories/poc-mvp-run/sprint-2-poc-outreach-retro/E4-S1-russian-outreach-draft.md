# Story E4-S1：生成俄语触达草稿

## 基本信息

- Epic：E4 AI 触达助手
- Sprint：Sprint 2 PoC 触达与复盘
- 优先级：P0
- 状态：Done

## 用户故事

作为客服，我希望 AI 根据审核模板生成俄语触达草稿，以便减少写话术时间。

## 业务价值

提高触达效率，同时控制不当承诺风险。

## 依赖

- E2-S2 AI 线索分级建议
- E9-S3 FAQ 与异议处理库初版

## 任务清单

- [x] 定义俄语触达模板。
- [x] 将 FAQ 中已审核内容作为可引用素材。
- [x] 为 B/C 级线索生成俄语触达草稿。
- [x] 输出风险提示和不确定信息。
- [x] 人工审核后才允许记录为已发送。
- [x] 保留话术版本和生成时间。

## 验收标准

- 话术必须基于审核模板和 FAQ。
- 话术不承诺最终价格、物流、清关、付款或交付周期。
- 邮件/官网表单触达包含拒绝联系路径。
- 输出风险提示。
- 人工确认后才能记录已发送。

## 非目标

- 不做自动社交私信。
- 不做自动加好友。

## QA / 风控检查

- [x] 每条外发话术必须人工审核。
- [x] 拒绝联系路径清晰。
