## 4. 风险与处理建议

- Forbidden 误放必须作为阻塞风险处理，修正过滤规则前不得进入 active_run。
- High 风险来源只允许进入人工复核，不得直接进入自动抽取。
- 证据缺失样本必须补充公开证据摘要，否则不得作为有效来源。
- 重复来源只保留对照标记，不写入 `lead_source_candidates`。
